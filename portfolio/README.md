## 💡 Technical Highlights

### 1. Geometric UI Alignment

Used custom calculated offsets for the timeline components to ensure perfect centering regardless of device size:


$$\text{Offset} = \left( \frac{\text{Circle Width}}{2} \right) + \left( \frac{\text{Line Width}}{2} \right)$$

### 2. Optical Text Alignment

Implemented `top-1` (4px) vertical offsets on absolute elements to match the **cap-height** of headings, ensuring a balanced visual flow.

### 3. Glow & Depth System

Developed a custom shadow system using `rgba` values to create depth in a dark-themed environment, moving away from "flat" design to a premium SaaS look.
