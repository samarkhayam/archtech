## 🛠️ Environment Setup

To run any of the tasks locally, ensure you have **Node.js** installed, then follow these steps:

1. Navigate to the specific task folder:
`cd task-folder-name`
2. Install the required dependencies:
`npm install`
3. Start the development server:
`npm run dev`
